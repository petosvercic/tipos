# tipos

Milestone 1 skeleton for:
- Next.js app router
- Supabase-backed eKeno result storage
- basic analysis summary API
- dashboard, history, analyzer, tip-panel pages

## Local start

1. Copy `.env.example` to `.env.local`
2. Fill the Supabase values
3. Run the SQL from `supabase/migrations/0001_init.sql`
4. Start with:

```bash
npm run dev
